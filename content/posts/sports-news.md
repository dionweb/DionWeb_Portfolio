# US Sport News

This website provides the latest US sports news using Gridsome and News API.

## Run locally

```
git clone https://github.com/dionweb/sports-news.git
cd sports-news
npm install

# Create .env file and put your API key.
# GRIDSOME_NEWS_API_KEY=

npm run develop
```

## Built With

-  Gridsome
-  News API
-  Tailwind CSS
